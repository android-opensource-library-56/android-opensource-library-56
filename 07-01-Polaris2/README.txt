Google Maps Android API v2を拡張した地図ライブラリ
サポートバージョン: Android 2.2 以上
License: Apache 2.0
タイプ: ライブラリプロジェクト
公式サイト: https://github.com/cyrilmottier/Polaris2

setup.shを実行するとPolaris2をcloneできます。
Polaris2をインポートすればサンプルコードをコンパイルできます。
動作にはGoogle Mpas API v2のAPIキーが必要です。以下のURLでAPIキーを取得して下さい。

Google Maps Android API v2
https://developers.google.com/maps/documentation/android/

詳細な導入手順や使い方については書籍を参照して下さい。
